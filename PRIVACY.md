# Privacy policy

Effective: 2026-07-18

The Jusigi plugin repository does not operate a hosted service, collect accounts, or receive user API credentials. The Skill generates or audits code in a repository the user authorizes.

Generated projects may connect directly from the user's GitHub Actions or local environment to services the user chooses, such as a broker, Telegram, a market-data provider, or OpenAI. Those services process data under their own terms and privacy policies. Users decide which integrations to enable and configure credentials directly in their own secret store.

Do not submit credentials, account identifiers, portfolio details, certificates, or personal data in public issues. Ordinary GitHub repository interactions are processed by GitHub under GitHub's policies.

Privacy questions may be submitted through a GitHub issue that contains no sensitive data.
